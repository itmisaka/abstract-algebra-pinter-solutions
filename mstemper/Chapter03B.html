Problem B — Groups on the Set R × R

For each of the following operations on the indicated subsets of R × R, which define groups? Which define Abelian groups?

  1. (a,b)*(c,d) = (ad+bc,bd) on the set R × (R - {0})

    (x*y)*z = ((a,b)*(c,d))*(e,f) = (ad+bc,bd)*(e,f) = (adf+bcf+bde,bdf)
    x*(y*z) = (a,b)*((c,d)*(e,f)) = (a,b)*(cf+de,df) = (adf+bcf+bde,bdf)
    The operation is associative.

    Let e = (e1, e2) and y = (y1, y2).
    Solving e*y = y gives: (e1, e2) * (y1, y2) = (y1, y2); (e1y2+e2y1, e2y2) = (y1, y2)
    Equating the first and second members of the ordered pairs gives:
    e1y2+e2y1 = y1 and e2y2 = y2
    Solving these for e1 and e2 gives e = (0,1).
    Let x = (x1,x2).
    Solving x*e = x gives: (x1, x2) * (e1, e2) = (x1, x2)
    Solving these for e1 and e2 again gives e = (0,1).
    Therefore, (0,1) is the identity element.

    Solving y'*y = (0,1) (and skipping the details) gives: y' = (-y1/y22, 1/y2).
    Similarly, solving x*x' = (0,1) gives: x' = (-x1/x22, 1/x2).
    Therefore, each element has an inverse.

    That this operation is commutative is immediate from the symmetry of its definition and the commutativity of addition and multiplication.

    This is an Abelian group.

  2. (a,b)*(c,d) = (ac,bc+d) on the set (R - {0}) × R

    (x*y)*z = ((a,b)*(c,d))*(e,f) = (ac,bc+d)*(e,f) = (ace,bce+de+f)
    x*(y*z) = (a,b)*((c,d)*(e,f)) = (a,b)*(ce,de+f) = (ace,bce+de+f)
    The operation is associative.

    Let e = (e1, e2) and y = (y1, y2).
    Solving e*y = y gives: (e1, e2) * (y1, y2) = (y1, y2); (e1y1,e2y1+y2) = (y1, y2)
    Equating the first members of the ordered pairs gives: e1y1 = y1; e1 = 1.
    Equating the second members of the ordered pairs gives: e2y1 = 0; e2 = 0.
    Let x = (x1,x2).
    Solving x*e = x gives: (x1, x2) * (e1, e2) = (x1, x2); (x1e1,x2e1+e2)
    Equating the first members of the ordered pairs gives:
    x1e1 = x1; e1 = 1.
    Equating the second members of the ordered pairs gives:
    x2 = x2e1+e2; x2 = x2e1+e2; e2 = 0.
    Therefore, (1,0) is the identity element.

    Solving y'*y = (1,0) gives: (y'1,y'2)*(y1,y2) = (1,0); (y'1y1,y'2y1+y2) = (1,0)
    Equating the first members of the ordered pairs gives:
    y'1y1 = 1; y'1 = 1/y1.
    Equating the second members of the ordered pairs gives:
    y'2y1+y2 = 0; y'2y1 = -y2; y'2 = -y2/y1; y' = (1/y1,-y2/y1).
    Similarly, solving x*x' = (1,0) and omitting the details gives: x' = (1/x1, -x2/x1).
    Therefore, each element has an inverse.

    (a,b)*(c,d) = (ac,bc+d)
    (c,d)*(a,b) = (ac,da+b)
    This operation is not commutative.

    This is a group, but it is not Abelian.

  3. (a,b)*(c,d) = (ac,bc+d) on the set R × R

    As in the previous exercise, this operation is associative and the operation has the same identity element (1,0).

    But it hasn't always an inverse because in the case of an element y = (0,y2) the eq. y'*y = (1,0) gives : (0,y'2)*(y1,y2) = (1,0) ; (0,y2) = (1,0) which is never satisfied.

    This is not a group.

  4. (a,b)*(c,d) = (ac-bd,ad+bc) on the set (R × R) - {0}

    (x*y)*z = ((a,b)*(c,d))*(e,f) = (ac-bd,ad+bc)*(e,f) = (ace-bde-adf-bcf,ade+bce+acf-bdf).
    x*(y*z) = (a,b)*((c,d)*(e,f)) = (a,b)*(ce-df,cf+de) = (ace-adf-bcf-bde,bce-bdf+acf+ade).
    The operation is associative.

    Let e = (e1,e2) and y = (y1, y2).
    Solving e*y = y gives: (e1y1-e2y2,e2y1+e1y2) = (y1,y2).
    Equating corresponding members of the ordered pairs gives:
    e1y1-e2y2 = y1
    e2y1+e1y2 = y2
    We need to deal with three separate cases:
    y1 = 0 and y2 ≠ 0
    y1 ≠ 0 and y2 = 0
    y1 ≠ 0 and y2 ≠ 0
    In each case, solving for e gives e = (1,0).
    Let x = (x1,x2).
    Solving x*e = x for the three equivalent cases also gives e = (1,0).
    Therefore, (1,0) is the identity element.

    Solving y'*y = (1,0) gives: (y'1,y'2)*(y1,y2) = (1,0); (y'1y1-y'2y2,y'1y2+y'2y1) = (1,0)
    Equating corresponding members of the ordered pairs gives:
    y'1y1-y'2y2 = 1
    y'1y2+y'2y1 =0
    Omitting all of the messy details, we end up with: y' = (y'1,y'2) = (y1/(y12+y22),-y2/(y12+y22)).
    The operation has an inverse.

    That this operation is commutative is immediate from the symmetry of its definition and the commutativity of addition and multiplication.

    This is an Abelian group.

  5. The same operation, but on the set R × R

    The operation is still associative.

    (1,0) is the identity element : we have already examined the cases where at least one of y1 or y2 is non-zero. All that remains is the case where y1 = 0 = y2. Although we cannot directly solve (e1,e2)*(0,0) = (0,0), we can see by inspection that e = (1,0) is a solution to this equation.

    However, each member of the inverse calculated in the previous exercise has a denominator of (y12+y22). Instead, when y1 = y2 = 0 , this denominator is zero. So the operation does not have an inverse for each element.

    As before, this operation is commutative.

    This operation is not a group.
